About BurnLab

BurnLab is designed to create a marketplace where fitness creators, trainers, and nutrition experts can sell products directly to users — while providing a clean user-friendly interface for buying, tracking orders, and managing personal support tickets.

This platform was built as part of a software engineering project. It includes account management, seller pages, order handling, and support tools for communication between users and administrators.

Thank you for using BurnLab!
